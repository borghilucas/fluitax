# Embedded Fonts

The PDF generator uses the following open licensed variable fonts downloaded from the Google Fonts repository:

- Inter Variable (SIL Open Font License 1.1)
- Roboto Mono Variable (Apache License 2.0)

Original sources:
- https://github.com/rsms/inter/releases
- https://github.com/google/fonts/tree/main/ofl/robotomono

Please reference the upstream repositories for full license text.
